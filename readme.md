# 🌟 OpenDSSMultiThread

Bem-vindo ao repositório **OpenDSSMultiThread**! Aqui você encontrará uma abordagem empolgante para executar simulações do OpenDSS de forma paralela, explorando todo o potencial do seu hardware. ⚡🧠

## 🚀 Motivação

Simulações elétricas podem ser **intensivas e demoradas**. Este projeto busca oferecer:
- 💨 **Aceleração**: execução paralela para reduzir o tempo de processamento.
- 🔁 **Flexibilidade**: scripts configuráveis para diferentes cenários.
- 📊 **Escalabilidade**: suporte a múltiplos casos e redes complexas.

- `src/`: código-fonte principal.
- `data/`: exemplos de arquivos de entrada.
- `docs/`: documentação complementar.

## ⚙️ Pré-requisitos

- 🐍 Python 3.10+
- 📦 Pipenv ou virtualenv
- 🔌 OpenDSS (versão compatível com automação)

Instale as dependências:
```bash
pip install -r src/requirements.txt
```

## ▶️ Como Executar

1. Clone o repositório:
    ```bash
    git clone https://github.com/pyyyter/OpenDSSMultiThread.git
    ```
2. Configure o ambiente.
3. Execute:
    ```bash
    python -m streamlit src/pages/app.py
    ```
